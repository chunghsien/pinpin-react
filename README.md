# Pinpin react

CMS & eCommrece 快速開發應用

## 詳見 MKDocs

 ** 安裝 `phthon` 及 `pip` ** <https://www.python.org/>

```
$ git clone https://github.com/chunghsien/pinpin_web_rad <project>

$ mkdocs new <project>

$ cd <project>

$ composer install

$ npm install

$ mkdocs serve
```
